# Data Dictionary

| Variable | Description |
|---|---|
| id | Individual respondent identifier |
| year | Survey year |
| age | Age of respondent |
| young | Binary indicator equal to 1 if respondent is aged 18–24 |
| post | Binary indicator equal to 1 for the post-NLW period |
| post_young | Interaction term between `post` and `young` |
| employed | Binary employment indicator based on ILO employment status |
| empstatus | ILO employment status |
| sex | Sex of respondent |
| marstatus | Marital status |
| ethnicity | Ethnicity category |
| education | Highest qualification |
| disability | Disability status |
| region | Region of work |
| industry | Industry sector |
| occupation | Occupation category |
| ftpt | Full-time or part-time work status |
| hourlypay | Hourly pay variable |
